# Hands-On Machine Learning

Repository ini berisi catatan, eksperimen, dan implementasi praktis Machine Learning yang gue pelajari secara hands-on. Fokus utamanya bukan cuma teori, tapi gimana konsep ML benar-benar dipakai lewat kode, eksperimen, dan studi kasus nyata.

Repo ini cocok buat mahasiswa, pemula ML, atau siapa pun yang lagi belajar Machine Learning dari nol sampai tahap implementasi model.

## Tujuan Repository

Tujuan utama repo ini adalah sebagai learning journal sekaligus playground Machine Learning. Setiap materi dicoba langsung lewat kode supaya konsepnya lebih kebayang, bukan cuma berhenti di rumus atau slide.

Selain itu, repo ini juga jadi arsip pembelajaran yang bisa dikembangin terus seiring bertambahnya topik dan eksperimen baru.

## Cakupan Materi

Materi di dalam repository ini mencakup dasar-dasar Machine Learning seperti supervised dan unsupervised learning, preprocessing data, training model, evaluasi performa, sampai penggunaan deep learning framework.

Beberapa eksperimen juga membahas aspek praktis seperti penggunaan GPU, optimasi training, serta distribusi training untuk skala besar.

## Tools & Teknologi

Sebagian besar eksperimen ditulis menggunakan Python dengan library Machine Learning yang umum dipakai di industri dan akademik. Framework utama yang digunakan meliputi TensorFlow dan Keras, serta library pendukung seperti NumPy, Pandas, dan Matplotlib untuk pengolahan serta visualisasi data.

Training model dilakukan baik di CPU maupun GPU, termasuk eksplorasi penggunaan Google Colab dan environment cloud untuk kebutuhan komputasi yang lebih besar.

## Struktur Repository

Struktur folder di repository ini disusun berdasarkan topik atau eksperimen. Setiap folder biasanya berisi notebook atau script Python, disertai penjelasan singkat mengenai tujuan eksperimen dan konsep yang dipelajari.

Pendekatan ini bikin materi lebih modular dan gampang diikuti satu per satu.

## Cara Menggunakan

Repository ini bisa langsung di-clone dan dijalankan secara lokal menggunakan Python. Disarankan menggunakan virtual environment agar dependensi lebih rapi.

Beberapa notebook juga bisa langsung dijalankan di Google Colab tanpa setup tambahan.

## Catatan

Repository ini bersifat ongoing dan akan terus diperbarui seiring proses belajar. Kode di dalamnya ditulis untuk tujuan pembelajaran, jadi masih sangat terbuka untuk perbaikan, optimasi, dan eksperimen lanjutan.

Kalau nemu bug, typo, atau punya saran pengembangan, feel free buat buka issue atau pull request.

## Lisensi

Repository ini dibuat untuk tujuan edukasi dan pembelajaran terbuka.
